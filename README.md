# ⚡ 3SR - استوديو معدل الإطارات

<div dir="rtl">

أداة متقدمة لمعالجة الفيديو تعمل محلياً على جهازك، توفر ميزتين رئيسيتين:

## 🎯 المميزات

### 🚀 1. زيادة الإطارات (Frame Interpolation)
- تحويل الفيديو من 30fps إلى 60fps أو أعلى
- استخدام خوارزميات متقدمة لإنشاء إطارات جديدة
- الحفاظ على جودة الفيديو الأصلية

### 🎭 2. تضليل المنصات (FPS Spoofing)
- تغيير معدل الإطارات الظاهري للفيديو
- جعل الفيديو يظهر كـ 30fps للمنصات وهو فعلياً 60fps
- مفيد لتجنب قيود بعض المنصات

## 📱 التوافق

- ✅ Mac / Linux / Windows
- ✅ يعمل بدون إنترنت بعد التثبيت
